# qiankun-demo介绍
微前端demo
- main 主应用，微前端基座
- app1 子应用，应用的vue框架
- app2 子应用，纯html项目

# 分支
main分支 通讯采用 qiankun API initGlobalState/onGlobalStateChange
dev分支 通讯基于 props 传参的模式
## Project setup
```
yarn
yarn start
```
Visit http://localhost:8000.
